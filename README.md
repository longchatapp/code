更新日期：2020年1月16日

概述
隐私保护与个人信息安全非常重要。我们制定本“隐私政策”并特别提示：希望你在使用“闪聊”前仔细阅读并理解本隐私政策，以便在获得充分信息的前提下，作出你的适当选择。

G1. 个人信息的梯度授权

a.我们会遵循隐私政策收集、使用你的信息，但不会仅因你同意本隐私政策而采用强制捆绑的方式一揽子收集个人信息。

b.为实现产品或服务的基本功能或遵循法律法规的规定，我们需要收集、使用相关的必要信息。由于缺少这些必要信息将导致基本功能无法实现或无法满足法律法规的要求，因此如果你拒绝提供必要信息，你将无法正常使用闪聊。我们会在隐私政策中逐项说明哪些是必要信息。

c.在基本功能之上，我们还提供了丰富的拓展功能，你可以选择使用并允许我们收集相关的信息。你可以拒绝提供这些信息，但是相关拓展功能可能无法使用或影响使用效果，但不会影响你使用闪聊的其他功能。


G2. 个人信息的自主可控

a.通讯录、精确地理位置、摄像头、麦克风、相册等敏感权限，不会默认开启，只有经过你的明示授权才会在为实现特定功能或服务时使用，你也可以撤回授权。

b. 闪聊可能会向你推荐可能认识的好友和感兴趣的信息，你可以随时选择关闭好友推荐功能。

c.你可以决定是否接受好友推荐、是否开启云存储信息功能，可以控制信息发布与展示的范围。

d.你可以选择开启或关闭聊天内容上传云端功能；你在云端存储的聊天内容将进行强加密存储，任何人无权非法访问、查看、获取、使用与披露。

e.如果你选择撤回授权或者将相关信息设置为非公开后，我们将不再收集和继续使用，除非获得你的再次授权或将相关信息设置为公开可见。


G3. 适用范围

本隐私政策适用于你通过闪聊移动客户端、PC端、供第三方网站或应用程序使用的闪聊软件开发工具包（SDK）和应用程序编程接口（API）等方式来访问或使用我们的产品或服务。

1.我们如何收集和使用个人信息

1.1 注册、登录、认证

1.1.1注册、登录

a.我们需要你提供手机号码，设置头像、昵称、密码以创建闪聊账户。手机号码、闪聊账户、密码是实现注册的必要信息，若你不提供可能无法正常使用闪聊。

b.你可以根据自身需求完善、修改头像、昵称，并自主选择是否填写性别、生日、地区、教育经历及个人简介以完善你的备注信息。你也可以修改上述信息，并随时选择将教育经历设置是否为公开。

1.1.2 认证

a.你使用特定功能或服务时，为遵循法律法规的要求或保障安全所必需，可能需要你选择基于身份证号码的认证。当你进行认证时需要你提供真实姓名和身份证号码。身份证号码是个人敏感信息，你如果拒绝提供将可能无法使用相关功能或获得相关服务，但不影响其他功能与服务的正常使用。

b.开通公共主页需要你授权手机号码以完成申请，个人手机号码是敏感个人信息，你拒绝提供可能将无法申请开通公共主页，但不影响你使用闪聊其他功能。

1.2 通讯联络

1.2.1 云端同步

a.当你使用闪聊进行通讯联络时，为了方便你在不同的设备上使用闪聊，帮助你快速备份、同步聊天记录，我们提供云端同步服务。

b.当你选择使用云端同步功能时，我们会收集聊天记录内容，因为收集是提供上述服务所必需。

c.为帮助你备份、存储文件，我们提供闪聊文件功能。当你使用该功能时，我们会收集你存储的文件，因为收集是提供上述服务所必须。

d.上传云端的信息和文件在传输、存储环节经过强加密保护，任何人无权非法访问、查看、获取、使用、修改、损毁、披露。

1.2.2 通讯联络

a.当你使用语音输入时，闪聊会收集你的语音信息，用于语音通讯。

b.在你发送音视频信息、使用语音与文字转换功能时，闪聊可能会向你请求授权摄像头和/或麦克风权限。这些权限是敏感权限，你可以拒绝提供且不影响使用闪聊的其他功能。

c.当你发送图文、音视频信息时，闪聊可能会向你请求授权相册权限。这一权限是敏感权限，你可以拒绝提供且不影响使用闪聊的其他功能。

1.3 社交互动

a. 你可以在闪聊中添加或删除好友，为帮你实现这一功能，我们会创建闪聊好友列表信息。闪聊好友列表是建立与管理好友关系所必需的必要信息。

b. 闪聊可能会根据用户授权的手机号码、通讯录、闪聊好友关系或你自主填写的教育信息为你发现好友。通讯录信息是个人敏感信息，你可以拒绝提供且不影响你正常使用闪聊的相关功能。请知悉，如果其他用户上传的通讯录中包含你的联系方式，系统也可能会基于该信息向你推荐好友。

c.当你撤回通讯录授权或将教育信息设置为“私密”后，系统将不再使用这些信息向你推荐好友，除非获得你的再次授权。如果你不希望推荐好友或者被推荐，可以在【我的】-【设置】-【隐私保护中心】中选择关闭。

d. 闪聊可能会基于你授权的GPS地理位置信息向你推荐附近的人。GPS地理位置信息是个人敏感信息，拒绝提供该信息将会影响该功能，但不影响你正常使用闪聊的其他功能。

e.当你选择使用扫码添加好友时，闪聊会向你请求摄像头权限，摄像头权限是敏感权限，拒绝提供仅会使你无法通过扫码途径添加好友，不影响其他功能的使用。

1.3.2群组互动

a.当你群功能时，我们会创建群聊列表及相关信息。

1.3.3浏览与发布

a.当你使用小组功能时，为实现功能所必需，我们会收集你发布的图文、音视频、收藏信息。

b.其他用户因使用我们的产品进行评论、留言、发布图文、音视频时，这些信息中可能会包含你的信息。

c.你发布信息并选择显示位置时，我们会请求你授权GPS地理位置权限，并收集与该服务相关的位置信息。精确地理位置信息属于个人敏感信息，拒绝提供仅会使你无法使用该功能，但不影响你正常使用闪聊的其他功能。

d.大多数手机等终端设备的相机或相机类App具有记录地理位置信息的功能，因此你拍摄照片或视频所在地的位置信息可能会在终端内保存。你未选择在动态发布时，闪聊无法也不会获取这些位置信息。你也可以自主选择在动态中发布时是否展示这些信息。如果你选择不展示，闪聊不会记录、保存这些位置信息。

1.4 推荐及展示

1.4.1收集的信息

a.用于信息展示的日志信息包括：

你操作、使用的行为信息：点击、关注、收藏、搜索、浏览、分享

地理位置信息：GPS信息、WLAN接入点、IP地址、蓝牙和基站等传感器信息

b.GPS地理位置是敏感个人信息，你拒绝提供我们将不会根据GPS信息向你推荐信息，且不会影响其他功能的正常使用。

1.5 搜索

搜索功能可以帮助你查询好友、群、小组和聊天记录，这是帮助闪聊用户实现社交互动所必需，当你使用搜索功能时，我们会收集你的搜索关键字信息、日志记录信息，用以优化搜索功能和向你推荐可能感兴趣的信息或服务。

1.6 保障安全正常运行

1.6.1安全保障功能信息

我们致力于为你提供安全、可信的产品与使用环境，提供优质而可靠的服务是我们的核心目标，因此安全保障功能是产品的基本功能。为实现安全保障功能所收集的信息是必要信息。

1.6.2设备信息与日志信息

a.当你使用闪聊及相关服务时，为了保障软件与服务的安全运行，闪聊会收集你的硬件型号、操作系统版本号、国际移动设备识别码、唯一设备标识符、网络设备硬件地址、IP 地址、WLAN接入点、蓝牙、基站、软件版本号、网络接入方式、类型、状态、网络质量数据、操作、使用、服务日志。

b.为了预防恶意程序及安全运营所必需，闪聊会收集安装的应用信息或正在运行的进程信息、应用程序的总体运行、使用情况与频率、应用崩溃情况、性能数据、应用来源。

c.我们可能使用你的账户信息、设备信息、服务日志信息以及我们关联公司、合作方在获得你授权或依法可以共享的信息，用于判断账户安全、进行身份验证、检测及防范安全事件。

1.8个人信息的处理与使用目的变更

1.8.1 设备信息、日志信息的处理使用

单独的设备信息、服务日志信息是无法识别特定自然人身份的信息。如果这些非个人信息与其他信息相结合用于识别特定自然人身份，或将其与个人信息结合使用，则在结合使用期间，这些信息将被视为个人信息，除取得你授权或法律法规另有规定外，我们会将这类信息做匿名化、去标识化处理。

1.8.2 收集、使用个人信息目的变更的处理

请你了解，随着我们业务的发展，可能会对闪聊的功能和提供的服务有所调整变化。当新功能或服务与本隐私政策所载明的应用场景相关时，收集与使用的个人信息将与原处理目的具有直接或合理关联。我们收集、使用你的个人信息在与原处理目的无直接或合理关联的场景下，会再次进行告知，并征得你的同意。

1.9 依法豁免征得同意收集和使用的个人信息

请你理解，在下列情形中，根据法律法规及相关国家标准，我们收集和使用你的个人信息无需征得你的授权同意：

a.与国家安全、国防安全直接相关的；

b.与公共安全、公共卫生、重大公共利益直接相关的；

c.与犯罪侦查、起诉、审判和判决执行等直接相关的；

d.出于维护个人信息主体或其他个人的生命、财产等重大合法权益但又很难得到本人同意的；

e.所收集的你的个人信息是你自行向社会公众公开的；

f.从合法公开披露的信息中收集的你的个人信息的，如合法的新闻报道、政府信息公开等渠道；

g.根据你的要求签订或履行合同所必需的；

h.用于维护软件及相关服务的安全稳定运行所必需的，例如发现、处置软件及相关服务的故障；

i.为合法的新闻报道所必需的；

j.学术研究机构基于公共利益开展统计或学术研究所必要，且对外提供学术研究或描述的结果时，对结果中所包含的个人信息进行去标识化处理的；

k.法律法规规定的其他情形。

2.我们如何共享、转让、公开披露个人信息

2.1共享

2.1.1 共享个人信息的原则

a.授权同意原则：向闪聊服务提供者以外的第三方共享、转让你的个人信息，需经过你的授权同意，除非共享、转让的个人信息是去标识化处理后的信息，且共享第三方无法重新识别此类信息的自然人主体。如果关联方、第三方使用信息的目的超越原授权同意范围，他们需要重新征得你的同意。

b.合法正当与最小必要原则：向关联方、第三方共享的数据必须具有合法正当目的，且共享的数据以达成目的必要为限。

c.安全审慎原则：我们将审慎评估关联方、第三方数据使用共享信息的目的，对这些合作方的安全保障能力进行综合评估，并要求其遵循合作法律协议。我们会对合作方获取信息的软件工具开发包（SDK）、应用程序接口（API）进行严格的安全监测，以保护数据安全。

2.1.2 实现功能或服务的共享信息

a.为保障你能正常使用闪聊中由我们的关联方或第三方提供的相关功能或服务，在你使用相关产品、开启或点击进入相关功能后，经你同意，我们可能会与关联方或第三方共享相关功能或服务所必需的信息。

b.为了帮你实现多端之间的交流沟通，你可以在关联方、第三方中使用闪聊。通信内容、好友关系、手机号码及相关敏感个人信息由闪聊收集，不会与关联方、第三方共享。

c.登陆第三方账号：你可以选择使用闪聊注册、登录第三方账号，经你自主选择同意后，闪聊将与第三方共享你在闪聊中的头像、昵称以及经你同意授权的其他信息。

d.地理位置服务：当你使用闪聊地理位置相关服务时，我们会将GPS信息与位置服务提供商进行共享以便可以向你返回位置结果。GPS信息是敏感个人信息，拒绝提供，仅会影响地理位置服务功能，但不影响其他功能的正常使用。

2.1.3实现安全与分析统计的共享信息

a.保障使用安全：我们非常重视账号与服务安全，为保障你和其他用户的账号与财产安全，使你和我们的正当合法权益免受不法侵害，我们和关联公司或服务提供商可能会共享必要的设备、账号及日志信息。

b.分析产品使用情况：为分析我们服务的使用情况，提升用户使用的体验，可能会与第三方共享产品使用情况（崩溃、闪退等）的统计性数据，这些数据难以与其他信息结合识别你的个人身份。

2.1.4 帮助你参加营销推广活动

当你选择参加我们及我们的关联方或第三方举办的有关营销活动时，可能需要你提供姓名、通信地址、联系方式、银行账号等信息。这些信息是敏感个人信息，拒绝提供可能会影响你参加相关活动，但不会影响闪聊的其他功能。只有经过你的同意，我们才会将这些信息与关联方或第三方共享，以保障你在联合活动中获得体验一致的服务，或委托第三方及时向你兑现奖励。

2.2转让

a.我们不会转让你的个人信息给任何其他第三方，除非征得你的明确同意。

b.随着我们业务的持续发展，我们将有可能进行合并、收购、资产转让，你的个人信息有 可能因此而被转移。在发生前述变更时，我们将按照法律法规及不低于本隐私政策所载明的安 全标准要求继受方保护你的个人信息，否则我们将要求继受方重新征得你的授权同意。

2.3 披露

a.我们不会对外公开披露其收集的个人信息，如必需披露时，在告知你公开披露的目的、披露信息类型等情况下，我们会征得你的同意。

b.在使用公开的群或小组时，你的头像、昵称可以被查看。

2.4 依法豁免征得同意共享、转让、公开披露的个人信息

请你理解，在下列情形中，根据法律法规及国家标准，我们共享、转让、公开披露你的个人信息无需征得你的授权同意：

a.与国家安全、国防安全直接相关的；

b.与公共安全、公共卫生、重大公共利益直接相关的；

c.与犯罪侦查、起诉、审判和判决执行等直接相关的；

d.出于维护你或其他个人的生命、财产等重大合法权益但又很难得到本人同意的；

e.你自行向社会公众公开的个人信息；

f.从合法公开披露的信息中收集个人信息的，如合法的新闻报道、政府信息公开等渠道。

3.我们如何存储个人信息

3.1存储地点

我们将境内收集的用户个人信息存储于中华人民共和国境内。目前我们不会将信息传输至境外，如果我们向境外传输，我们将会遵循国家有关规定并征求你的同意。

3.2存储期限

我们将为实现目的所必需的时间保留你的个人信息：

a.手机号码：在你使用期间，我们需要一直保存你的手机号码，以保证你正常使用该服务，当你注销闪聊帐户后，我们将删除或匿名化处理相应的信息；

4.我们如何保护个人信息的安全

a.我们非常重视你个人信息的安全，将努力采取合理的安全措施（包括技术方面和管理方面）来保护你的个人信息，防止你提供的个人信息被不当使用或未经授权的情况下被访问、公开披露、使用、修改、损坏、丢失或泄漏。

b.我们会使用不低于行业同行的加密技术、匿名化处理等合理可行的手段保护你的个人信息，并使用安全保护机制防止你的个人信息遭到恶意攻击。

c.我们会建立专门的安全部门、安全管理制度、数据安全流程保障你的个人信息安全。我们采取严格的数据使用和访问制度，确保只有授权人员才可访问你的个人信息，并适时对数据和技术进行安全审计。

d.尽管已经采取了上述合理有效措施，并已经遵守了相关法律规定要求的标准，但请你理解，由于技术的限制以及可能存在的各种恶意手段，在互联网行业，即便竭尽所能加强安全措施，也不可能始终保证信息百分之百的安全，我们将尽力确保你提供给我们的个人信息的安全性。

e.你知悉并理解，你接入我们的服务所用的系统和通讯网络，有可能因我们可控范围外的因素而出现问题。因此，我们强烈建议你采取积极措施保护个人信息的安全，包括但不限于使用复杂密码、定期修改密码、不将自己的账号密码等个人信息透露给他人。

f.我们会制定应急处理预案，并在发生用户信息安全事件时立即启动应急预案，努力阻止该等安全事件的影响和后果扩大。一旦发生用户信息安全事件（泄露、丢失等）后，我们将按照法律法规的要求，及时向你告知：安全事件的基本情况和可能的影响、我们已经采取或将要采取的处置措施、你可自主防范和降低风险的建议、对你的补救措施等。我们将及时将事件相关情况以推送通知、邮件、信函、短信等形式告知你，难以逐一告知时，我们会采取合理、有效的方式发布公告。同时，我们还将按照相关监管部门要求，上报用户信息安全事件的处置情况。

5.管理你的个人信息

我们非常重视你对个人信息的管理，并尽全力保障你对于信息的访问、修改、删除以及撤回授权同意的权利。

5.1访问、更正、删除你的个人信息

你可以通过以下方式管理你的信息：

a.当你完成闪聊的账号注册、登录并进行合理和必要的身份验证后，你可以在【我的】中查阅、修改、删除你提交给闪聊及相关服务的个人信息（包括删除你的聊天记录，修改昵称、头像）。

请你理解，特定的业务功能和服务将需要你的信息才能得以完成，当你撤回同意或授权后，我们无法继续为你提供撤回同意或授权所对应的功能和服务，也不再处理你相应的个人信息。但你撤回同意或授权的决定，不会影响我们此前基于你的同意或授权而开展的个人信息处理。

5.3注销你的账号

a.你可以在【我的】-【设置】-【账号与安全】中选择注销账号。

b.在你注销账号前，我们将验证你的个人身份、安全状态、设备信息、日志信息。

c.注销账号的行为是不可逆的行为，当你注销账号后，我们将删除有关你账号的相关信息或进行匿名化处理，但法律法规另有规定的除外。

5.4访问隐私政策

你可以在注册页面，或者登录闪聊客户端在【我的】-【关于闪聊】查看本隐私政策的全部内容。

5.5响应你的请求

5.5.1 响应请求

a.你可按照我们公示的制度进行投诉或举报。如果你认为你的个人信息权利可能受到侵害，或者认为收集你的个人信息违反法律规定或者双方约定，你可以通过点击【我的】-【帮助与反馈】-【账号与安全】，进入用户反馈界面与我们联系。

b.为保障安全，我们可能会先要求你验证自己的身份，然后再处理你的请求。

c.对于与你的身份不直接关联的信息、无端重复信息，或者需要过多技术手段、给他人合法权益带来风险或者不切实际的请求，我们可能会予以拒绝。

5.5.2响应请求的例外

在以下情形中，按照法律法规要求，我们将无法响应你的请求：

a.与国家安全、国防安全有关的；

b.与公共安全、公共卫生、重大公共利益有关的；

c.与犯罪侦查、起诉、审判和执行判决等有关的；

d.有充分证据表明个人信息主体存在主观恶意或滥用权利的；

e.响应请求将导致你或其他个人、组织的合法权益受到严重损害的；

f.涉及商业秘密的。
